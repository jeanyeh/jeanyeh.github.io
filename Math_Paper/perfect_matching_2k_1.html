Perfect Matching of 2k

Perfect Match of 2k, with 2k at the end

定義：

  • Perfect Matching of [2n]:
    A set partition of [2n] with blocks (disjoint nonempty subsets) of size exactly 2. Let $\mathit{M}_{2n}$ be the set of matchings of [2n], and let $M \in \mathit{M}_{2n}$. The standard form of $M$ is a list of blocks $\{(i_1, j_1), (i_2, j_2), ..., (i_n, j_n) \}$ such that $i_r < j_r$ for all $1\leq r\leq n$ and $1=i_1 < i_2 < ... < i_n$.

  • Perfect Matching of [2k] with 2k at the end:
    For above $M$ must have $j_k=2k$.

  • $ee(\rho)$ = #$\{i \mid \rho(i) \mbox{ is even and } \rho(i+1) \mbox{ is even}\}$

  • $eo(\rho)$ = #$\{i \mid \rho(i) \mbox{ is even and } \rho(i+1) \mbox{ is odd}\}$

  • $oe(\rho)$ = #$\{i \mid \rho(i) \mbox{ is odd and } \rho(i+1) \mbox{ is even}\}$

  • $oo(\rho)$ = #$\{i \mid \rho(i) \mbox{ is odd and } \rho(i+1) \mbox{ is odd}\}$
