Burhat Order

Burhat Order (back to Data page)

定義：

$w$ is one line permutation in $\mathcal{S}_n$
  • $l_O(w)= \Big| \{ ( w(i), w(j) ) \mid1\leq i < j \leq n, w(i)> w(j), i \not\equiv j ~ (mod ~2) \} \Big|$

  • $l_E(w)= \Big| \{ ( w(i), w(j) ) \mid1\leq i < j \leq n, w(i)> w(j), i \equiv j ~(mod ~2) \} \Big|$

  • $l_{ex}(w)= \Big| \{ ( w(i), w(j) ) \mid1\leq i < j \leq n, w(i)> w(j), i \equiv 0 ~(mod ~2) \} \Big|$

  • $l_{xe}(w)= \Big| \{ ( w(i), w(j) ) \mid1\leq i < j \leq n, w(i)> w(j), j \equiv 0 ~(mod ~2) \} \Big|$

  • $l_{ee}(w)= \Big| \{ ( w(i), w(j) ) \mid1\leq i < j \leq n, w(i)> w(j), i \equiv j \equiv 0 ~(mod ~2) \} \Big|$
