permB_exc_aexc_fix

Permutaion of type B, in cycles, with $exc, aexc, fix$ (back to Data page)

定義：

  • Permutaion of type B：

  • $exc$($\rho$) = # $\{i \mid \rho(|\rho(i)|) > \rho(i) \}$

  • $aexc$($\rho$) = # $\{i \mid \rho(|\rho(i)|) < \rho(i) \} \cup \{i \mid \rho(i) = \bar{i}\}$

  • $fix$($\rho$) = # $\{i \mid \rho(i) = i \}$

  • 在本網頁的例子中，$\bar{i}$ 寫作 $-i$

$\pi(0)=0$, basc, des$_B$, suc
